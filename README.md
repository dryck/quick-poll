# 🎫 Exit Ticket Express

Быстрая проверка понимания в конце урока с слепым голосованием.

## 🌐 Live Demo
https://dryck.github.io/quick-poll/

## ✨ Новые фичи

### 🎨 Современный дизайн
- Анимированные градиенты
- 3D-кнопки с эффектом нажатия
- Стеклянный эффект (glassmorphism)
- Плавные анимации

### 🙈 Слепое голосование
- Ученики видят только кнопки
- Результаты скрыты до нажатия кнопки
- Учитель видит: "Проголосовало: X / Y"
- Прогресс-бар голосования
- Авто-открытие когда все проголосовали

## 🚀 Как использовать

### 1. Настройка (10 сек)
- Выберите тип: 🟢🟡🔴 😊😐😢 ⭐⭐⭐⭐⭐ 🐢🐕🐇 💪🤷🙈
- Введите вопрос (или оставьте)
- Укажите количество учеников

### 2. Голосование
- Поставьте ноутбук на видном месте
- Дети подходят и жмут кнопки
- Вы видите только счётчик

### 3. Результаты
- Жмёте "Показать результаты"
- Красивые анимированные бары
- Распределение голосов

- "Тема сегодня была понятна?"
- "Как вы себя чувствуете перед контрольной?"

- "Динг" при голосовании
- "Та-да-да" при открытии результатов
