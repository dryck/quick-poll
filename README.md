# 🎫 Exit Ticket Express

Быстрая проверка понимания в конце урока. Один экран, большие кнопки, мгновенные результаты.

## 🌐 Live Demo
https://dryck.github.io/quick-poll/

## ✨ Как использовать

### 1. Выберите тип (1 клик)
- 🟢🟡🔴 Понимание (Да/Частично/Нет)
- 😊😐😢 Настроение
- ⭐⭐⭐⭐⭐ Рейтинг 1-5
- 🐢🐕🐇 Темп (медленно/нормально/быстро)
- 💪🤷🙈 Уверенность

### 2. Введите вопрос
Или оставьте предложенный

### 3. Нажмите "Начать голосование"

### 4. Дайте ноутбук ученикам
Дети подходят и жмут свои кнопки

### 5. Смотрите результаты
Счётчик показывает сколько проголосовало, графики обновляются в реальном времени

## 🎯 Преимущества

- ⚡ **Быстро** — 10 секунд на подготовку
- 👆 **Просто** — большие кнопки для детей
- 📊 **Наглядно** — результаты видны сразу
- 🔊 **Звук** — подтверждение при нажатии
- 🎮 **Игровое** — эмодзи вместо скучного текста

## 💡 Идеи для вопросов

- "Тема сегодня была понятна?"
- "Как вы себя чувствуете перед контрольной?"
- "Оцените нашу групповую работу"
- "Что было самым сложным сегодня?"

## 🛠️ Технические детали

- Работает оффлайн
- Сохраняет данные в браузере
- Fullscreen режим для проектора
- Адаптировано для планшетов
